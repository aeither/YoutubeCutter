## Downloaded file folder hierarchy 

Elements are sorted by types (Audio / Video) :

  - Videos
    - videoX
    - videoY
    - playlistX
      - videoXinPlaylistX
  - Audios
    - audioX
    - audioY
    - playlistX
      - audioXinPlaylistX
